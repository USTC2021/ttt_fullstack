- 1.2 分析javaScript
    1. javaScript 函数 
        基于词法作用域的顶级对象
        (js作用域链)词法作用域 = 静态作用域
        词法作用域：在函数声明时确定
        动态作用域：在函数调用时确定
    2. javaScript 弱类型语言
        弱类型语言：编译器不能检测出类型错误
        强类型语言：编译器在编译时检测错误
    4. javaScript 有对象字面量表示法
    5. javaScript 原型继承
